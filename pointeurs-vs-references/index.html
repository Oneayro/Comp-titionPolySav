Pointeurs

En C++, un pointeur est une variable contenant l’adresse mémoire d’une autre variable. Il offre un contrôle très bas niveau, permettant de manipuler directement la mémoire.

Les pointeurs doivent être déréférencés avec * pour accéder à la valeur pointée, et utilisent & pour obtenir l’adresse d’une variable. 

int i = 5;

// p stocke l'adresse de i
int* p = &i;

// déréférencement
*p = 10;     // i devient 10
  • peut être nullptr,
  • peut changer de cible,
  • autorise l'arithmétique (+1, -1…),
  • permet d’implémenter des structures comme les listes chaînées et arbres.

Références

Une référence est un alias vers une variable existante. Contrairement au pointeur, elle ne possède pas sa propre adresse et doit être initialisée immédiatement.

Une référence est plus sûre : elle ne peut jamais être null, et ne peut jamais changer d’objet référencé. Le compilateur gère automatiquement l’indirection. 

int i = 5;

// alias de i
int& r = i;

r = 10;  // i devient 10
  • initialisation obligatoire,
  • jamais null,
  • pas de réassignation,
  • syntaxe simple et lisible.
